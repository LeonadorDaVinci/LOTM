欢迎测试该控制台游戏，以下是几个忠告:
不要冒险尝试死亡，这会重置你的所有游戏进度
不要关闭控制台进程，这也会重置你的所有游戏进度
家中的钱或随身携带的钱不够平衡会导致意外发生

关于游戏内的互动语句
通常会提供有多个选项以供选择，如填写关键字以外的回答将会默认选择第一个选项。

数据查询器，随时可以使用
命令列表
default name - 获取初始姓名
default address - 获取初始地址

current address - 获取当前地址

home - 查看是否在家中

total money - 总金钱（包括家中的和随身携带的）
current money - 当前随身携带的金钱
get money amount=[amount] -从家中取钱
save money amount=[amount] -往家中存钱
more info -获取更多信息（只有在互动语句后方出现 /MORE INFORMATION AVAILABLE 时可执行）

music shut down - 立刻停止音乐进程
music restart - 重启音乐（音乐采用指环王主题曲。为什么不采用诡秘之主主题曲？因为我觉得那一首不太符合氛围。）

【开发命令-如果需要帮助】
set music path=[path] - 经测试有部分电脑将无法加载背景音乐，故设置该指令以便玩家自定义音乐路径（如音乐正常播放不建议更改。该命令仅供测试用途。）
-当终端提示：音乐路径设置为：*** 时，只需再次执行 music restart 即可加载音乐（如报错则是路径设置错误）
如何查看文件绝对路径？
-请自行google
-主操作系统大都分为三类，Mac，Windows，Linux，各自有查看系统文件绝对路径的快捷方法

推荐设定
北大陆，鲁恩王国，廷根/贝克兰德，平民/中产
-为什么推荐该配置？
-因为其它的选项没有完善，如果选择其它将会无法继续进程

启动方法
确保电脑已配置JAVA环境（推荐JDK-17）
打开终端/命令行，如果不知道如何打开请自行google
在终端控制台进入主目录（*Jlotm），如果不知道如何进入目录请自行google
输入命令：（可复制）
java -jar MavenLOTM.jar
